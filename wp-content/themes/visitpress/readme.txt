== Description ==

VisitPress is an easy customizable WordPress blog theme. It is a fully responsive theme that allows for easy viewing on any device.

== Installation ==

You can install the theme through the WordPress installer under Themes-Install themes by searching for it.
Alternatively you can download the file from here, unzip it and move the unzipped contents to the wp-content/themes folder
of your WordPress installation. You will then be able to activate the theme.

== Documentation ==

The complete documentation of VisitPress names "documentation.html" and is located in "docs" folder in this theme main folder. It is also accessable via Theme Options Panel after installing and activating this theme. 

== Other notes ==

If you are using WordPress 3.4.2 or older, please download Cyclone Slider 2 Plugin in version 2.6 or older for displaying slideshows (or update your WordPress version). The current version of Cyclone Slider 2 requires WordPress 3.5 or higher.
